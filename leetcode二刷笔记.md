```
    int removeDuplicates(vector<int>& nums) {
        int count = 0, n = nums.size();
        for (int i = 1; i < n; i++) nums[i] == nums[i-1] ? count++ : nums[i-count] = nums[i];
        return n - count;
    }
```

### 50. Pow(x, n)(不会)
好题三刷
1.对INT_MIN要单独处理
```
unsigned int nn;
if (n == INT_MIN) nn = (unsigned int)INT_MAX + 1;
else nn = (n >= 0 ? n : -n);
```
2.公式要背牢
```
double Pow(double x, unsigned int n)
{
    if (n == 0) return 1.0;
    if (n == 1) return x;
    double r = Pow(x, n>>1);
    r *= r;
    if (n & 0x01) r *= x;
    return r;
}
```
